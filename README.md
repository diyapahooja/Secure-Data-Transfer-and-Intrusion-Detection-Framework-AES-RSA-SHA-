# CipherVault 🔐
Secure Data Transfer and Intrusion Detection Framework (SDTIDF)

## Features
- **AES-256** for data confidentiality (Symmetric)
- **RSA-2048** for secure key exchange (Asymmetric)
- **SHA-256** for data integrity verification
- **RSA-PSS** for digital signatures (Non-repudiation)
- **Intrusion Detection** (Brute-force login prevention)
